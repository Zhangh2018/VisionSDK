Bilinear interpolation kernel metadata with direct inputs and simulated indirect chunks. Interpolates the pixels in one chunk of unsigned 16bit image according to the chunk related xoffs and yoffs Outputs 16bit signed interpolation image of same size as the input image. Out(x,y) = [(1-xOffs)(1-yOffs)In(x,y) + xOffs(1-yOffs)In(x+1, y) + yOffs*(1-xOffs)*In(x, y+1) + xOffs*yOffs*In(x+1,y+1)]/(quotx*quoty)
