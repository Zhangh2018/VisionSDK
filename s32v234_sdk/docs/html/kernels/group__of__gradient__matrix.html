Accumulate the elements of the gradient matrix kernel metadata.

Calculate the sum of the chunk elements of each of the Lucas-Kanade Gradient Matrix elements over a defined neighborhood specified by (ACCUM_TILE_SIZE_X, ACCUM_TILE_SIZE_X) Outputs the three vectors of accumulated GradX^2, GradY^2, GradX*GradY and the vector of dot inverse of the determinant of previous three(i.e. detG = accGradXSqr*accGradYSqr - accGradXY^2, invDetG = 1/detG)

Port GRADIENT_MATRIX_KN_GRAD_XYDefine for name of GradX*GradY matrix input image (signed 32bit),
Port GRADIENT_MATRIX_KN_ACC_GRAD_X_SQRDefine for name of accumulated Square of GradX output vector (unsigned 32bit).
Port GRADIENT_MATRIX_KN_ACC_GRAD_Y_SQRDefine for name of accumlated Square of GradY output vector (unsigned 32bit)
Port GRADIENT_MATRIX_KN_ACC_GRAD_XYDefine for name of accumulated GradX*GradY output vector (signed 32bit),
