Sum of values over one patch the input image is a SAT image (i.e. integral computed with the sat32() function.

Applies a box filter (== sum over that patch) to the image using its summed area table (integral image).

out(i,j) = in(i - box_size - 1, j - box_size - 1) + in(i + box_size, j + box_size) - in(i - box_size - 1, j + box_size) - in(i + box_size, j - box_size - 1)

Computes the signed sum of values over one patch, by using the SAT (integral) image computed previously. Does not need indirect inputs.
