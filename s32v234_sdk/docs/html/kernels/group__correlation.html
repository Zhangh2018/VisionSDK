General correlation kernel metadata. General correlation of input unsigned 8bit image with a 1D or 2D filter. Outputs correlation result (signed 16bit).

Warning
The filter width and height have to be defined in the macro definitions FW and FH for the currently used kernel!
Parameters
CORR_KNDefine for Kernel name
5Number of ports
Port CORR_Kernel_InputDefine for name of input image (unsigned 8bit). The spatial dependencies of this port - HFW and HFH - are half the filter width/height. They are defined in the header of this file as HFW FW>>1 and HFH FH >> 1
Port CORR_Kernel_OutputDefine for name of correlation result (signed 16bit)
Port CORR_Kernel_FilterDefine for name of filter coefficients the input has to be convolved with (signed 16bit)
Port CORR_Kernel_FilterScaleDefine for name of the scalar value of the normalization factor used for the filter (signed 16bit)
Port CORR_Kernel_FiltSymmFlDefine for name of the scalar value of the symmetry flag defined for this filter as it is specified in "symmetry_flags.h" file (unsigned 16bit)
