19 /*!*********************************************************************************
20 * @file
21 * @brief \ref secHarrisCorner "Harris corner detection" implementation for the APU.
22 *
23 * \section secHarrisCorner Harris Corner Detection
24 *
25 * Harris corner detection is a technique that identifies corners present in an image (denoted \em I below).
26 * The technique first calculates the weighted sum of squared differences between \em I(u,v) and \em I(u+x,v+y).
27 *
28 * \image html harris0.svg
29 * \image latex harris0.pdf
30 *
31 * By a Taylor expansion, it can be approximated as follows.
32 *
33 * \image html harris1.svg
34 * \image latex harris1.pdf
35 *
36 * which can be written in matrix form
37 *
38 * \image html harris2.svg
39 * \image latex harris2.pdf
40 *
41 * where \em A is a second-moment matrix,
42 *
43 * \image html harris3.svg
44 * \image latex harris3.pdf
45 *
46 * This matrix is a Harris matrix.
47 * \em Ix and \em Iy are the partial derivatives of \em I, and those are obtained by Sobel operation.
48 * Elements of a Harris matrix is averaged by Box blur operation (denoted as "<>").
49 * Then corner score is calculated as follows.
50 *
51 * \image html harris4.svg
52 * \image latex harris4.svg
53 *
54 ***********************************************************************************/
