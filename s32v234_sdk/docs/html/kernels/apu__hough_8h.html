Store the lines for which the accumulator is above a threshold and using non-maxima suppression.

Returns
Line count in CMEM.
Parameters
linesLine destination buffer. Each line is described by 32 bits. The first 8 bits are the theta ID. The next 12 bits are the rho ID. The last 12 bits are the accumulator value.
accumulatorAccumulator data (rho x theta space).
thetaIndexAngle index for each CU.
rhoCountNumber of rho coordinates.
thetaCountNumber of angles to detect. This must not exceed cuCount-2 if non-maxima suppression between angles is enabled, otherwise it must not exceed cuCount.
thresholdAccumulator threshold.
maxLinesPerCUMaximum lines that can be returned for each CU.
nonMaxSupFlagNon-maxima suppression flag.
cuCountNumber of computational units.
